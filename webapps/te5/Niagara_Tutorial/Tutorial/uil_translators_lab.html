UIL Translators - Lab

 

Exercise 1

  • Add a uil translator called "test" to your list with the uil create command.
  • Confirm that the translator is in the list with the uil list command.
  • Select the new translator with the uil set index command
  • Remove the translator with the uil remove command

Exercise 2 (Safari Users Only)

  • Copy the files work.exe, reserve.img, flash.img, and mfgtest.dll to the C:/cil directory
  • Start Niagara through the Niagara (SPTI) path
  • Create an instance of simscsi that automatically starts work using the command: uil create simscsi 1
  • Select the new translator with uil set index <index of simscsi>
  • Send an inquiry to the simulator
  • Unload the simulator and kill work.exe with the command: uil remove <index of simscsi>
